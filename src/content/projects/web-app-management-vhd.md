## Overview

Aplikasi manajemen properti berbasis web yang dirancang khusus untuk memenuhi kebutuhan operasional **Villa Hotel Dieng**. Sistem ini menyederhanakan seluruh alur kerja perhotelan—mulai dari pencatatan reservasi tamu, pemantauan ketersediaan kamar secara *real-time*, penagihan (*billing*), hingga manajemen kebersihan kamar (*housekeeping*). Dengan antarmuka yang intuitif, manajemen dapat memonitor performa bisnis kapan saja untuk memastikan pelayanan tamu yang maksimal.

## Tech Stack

- **Backend:** ASP.NET Core, C#
- **Database:** SQL Server
- **Frontend:** TailwindCSS, Vanilla JavaScript
- **Architecture:** MVC (Model-View-Controller) / RESTful API

## Key Features

- **Interactive Dashboard:** Pemantauan status kamar (Tersedia, Terisi, Dibersihkan, Perbaikan) dan matrik okupansi harian dalam satu layar.
- **Reservation Management:** Modul *check-in/check-out* yang mulus dengan pencatatan data tamu dan riwayat transaksi yang terstruktur.
- **Housekeeping Tracking:** Pembaruan status kebersihan kamar secara *real-time* untuk sinkronisasi antara staf lapangan dan resepsionis.
- **Automated Reporting:** Pembuatan laporan keuangan, tingkat hunian, dan analitik tamu dengan opsi *export* data langsung ke format PDF dan Excel.
- **Role-Based Access Control:** Keamanan sistem dengan pembagian hak akses (Admin, Front Desk, Housekeeping, Manager).
