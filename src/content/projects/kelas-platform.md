## Overview

Berkontribusi pada pengembangan dan optimasi platform e-learning Kelas.com — salah satu platform belajar online terbesar di Indonesia. Fokus pada peningkatan performa, stabilitas sistem, dan pengembangan fitur baru.

## Tech Stack

- **Backend:** PHP, Laravel
- **Database:** MySQL
- **Frontend:** TailwindCSS, JavaScript, Blade Templates
- **DevOps:** CI/CD, monitoring tools

## Key Achievements

- Mengurangi waktu siklus pengembangan fitur sebesar **25%** dengan komponen kode yang dapat digunakan ulang
- Mencapai **99.95% uptime** melalui pemantauan proaktif dan penyelesaian 50+ bug kritis per kuartal
- Meningkatkan skor Lighthouse dari **54 → 90/100** melalui optimasi aset
- Mengurangi waktu respons server sebesar **60%** melalui optimasi query database
