## Overview

**Website Villa Hotel Dieng** adalah platform pusat yang dirancang khusus untuk mempermudah proses pemesanan vila dan hotel. Tujuan utama situs web ini adalah untuk menyederhanakan dan mengoptimalkan cara pengguna mengeksplorasi, memilih, dan memesan akomodasi sesuai dengan preferensi serta kebutuhan mereka. 

Sebagai etalase digital utama bagi Villa Hotel Dieng, platform ini menghadirkan antarmuka visual yang menawan dan ramah pengguna (*user-friendly*). Pengunjung dapat dengan mudah melihat detail fasilitas kamar, mengecek ketersediaan secara *real-time*, hingga menyelesaikan transaksi pembayaran dengan aman hanya dalam beberapa klik.

## Tech Stack

- **Backend:** ASP.NET Core, C#
- **Database:** SQL Server
- **Frontend:** TailwindCSS, Vanilla JavaScript
- **Architecture:** MVC (Model-View-Controller) / RESTful API

## Key Features

- **Smart Search & Filtering:** Pencarian akomodasi yang intuitif berdasarkan tanggal menginap, jumlah tamu, rentang harga, dan preferensi fasilitas (misal: kolam renang, pemandangan gunung).
- **Real-Time Availability:** Sinkronisasi data ketersediaan kamar yang instan, memberikan kepastian kepada tamu saat melakukan pemesanan tanpa takut terjadi *double-booking*.
- **Seamless Checkout & Payment:** Integrasi *payment gateway* terenkripsi untuk memfasilitasi berbagai metode pembayaran yang aman, cepat, dan mudah.
- **Guest Portal:** Dasbor personal bagi pengguna untuk melacak status reservasi, mengunduh bukti pemesanan (*e-voucher*), dan mengelola riwayat perjalanan mereka.
- **Responsive & Immersive Design:** Galeri foto beresolusi tinggi dengan tata letak yang beradaptasi sempurna di layar *smartphone*, tablet, maupun komputer desktop.
