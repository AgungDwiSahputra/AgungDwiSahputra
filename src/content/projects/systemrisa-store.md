## Overview

Platform e-commerce dan sistem booking yang dibangun secara mandiri sebagai founder dan full-stack developer. Sistem dirancang untuk berjalan secara self-sustaining dengan otomatisasi maksimal, mengurangi ketergantungan pada intervensi manual.

## Tech Stack

- **Backend:** PHP, Laravel
- **Frontend:** TailwindCSS, JavaScript
- **Database:** MySQL
- **Tools:** Canva, Adobe Firefly (konten visual)

## Key Achievements

- Membangun sistem website yang mengelola **booking, payment, dan update konten** secara otomatis
- Mengurangi biaya operasional sebesar **50%** melalui otomatisasi workflow
- Meningkatkan visibilitas SEO sebesar **65%** dengan structured data dan optimasi kecepatan
- Meningkatkan performa halaman **55%** (3.2s → 1.4s) menggunakan kompresi gambar, caching, dan CDN
- Meningkatkan skor mobile responsiveness dari **68 → 89/100**
