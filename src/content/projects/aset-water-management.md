## Overview

Sistem informasi geografis (GIS) untuk manajemen aset infrastruktur jaringan air bersih. Memungkinkan tim lapangan dan manajemen memonitor kondisi aset, jadwal perawatan, dan distribusi jaringan pipa secara visual di atas peta interaktif.

## Tech Stack

- **Backend:** ASP.NET Core, C#
- **Database:** SQL Server
- **Frontend:** TailwindCSS, JavaScript, Leaflet.js
- **GIS:** Spatial data management, shapefile integration

## Key Features

- Peta interaktif dengan layer aset (pipa, pompa, reservoir, meter air)
- Dashboard monitoring kondisi aset real-time
- Manajemen jadwal pemeliharaan preventif dan korektif
- Laporan aset dengan filter lokasi, kategori, dan status
- Export data ke PDF dan Excel
